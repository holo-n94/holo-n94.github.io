Sha1coin対応sgminer

概要

Sha1coinにも対応した所謂GPU用マイナーです。

グラフィックカードやCPUに統合されたグラフィック処理部分を利用することで、 Sha1coinの採掘やトリップ検索をCPUのみの場合より効率よく行えます。

利用にはOpenCLが動作する環境が必要となります。

ダウンロード

  • Windows用バイナリ : sph-sgminer-20150725-win32bin.zip
  • ソースコード : sph-sgminer-20150725.zip
  • GitHub : holo-n94/sph-sgminer

ビルド例

./autogen.sh
./configure
make

依存関係やオプションについてはREADME.mdを参照してください。

実行例

Windows
sgminer.exe -k sha1coin -o <host>:<port> -u <user> -p <password> -I 20 -g 1 -w 64 --trip GPGPU/

Linux等
GPU_MAX_ALLOC_PERCENT=100 ./sgminer -k sha1coin -o <host>:<port> -u <user> -p <password> -I 20 -g 1 -w 64 --trip GPGPU/
